== Description ==

Allows authors to geocode their posts & pages using the upcoming [Geolocation API](http://dev.w3.org/geo/api/spec-source.html "") and display the location using [Google Maps](http://maps.google.com/ "").

It also supports the [geo-location-javascript library](http://code.google.com/p/geo-location-javascript/ "") and the [MaxMind GeoIP Javascript Service](http://www.maxmind.com/app/javascript_city "") for backwards compatibility.

== Installation ==

1. Upload `/author-geolocation/` to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu.
1. (Optional) Specify your settings through the settings menu.
1. Start using it.
